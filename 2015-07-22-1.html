vimでブラウザ操作するハメになる

Date 2015年07月22日(水) Tags vim

CentOS6のはなし 諸事情により、vimからテキストブラウザを開きたくなりました

  • lynxを入れる
yum -y install lynx
  • /etc/lynx.confの変更
CHARACTER_SET:utf-8
PREFERRED_LANGUAGE:ja

CHARACTER_SETははじめらutf-8だった

  • vim-refのインストール
NeoBundle 'thica/vim-ref'

.vimrcに書いて、いつもの :NeoBundleInstall

  • ここまで書いて、lynxがめんどくさそうだったので、w3mに方向転換
yum -y install w3m
  • w3m.vimのインストール
NeoBundle 'yuratomo/w3m.vim'

.vimrcに書いて、いつもの :NeoBundleInstall

これで :W3cでグーグル検索できるようになった。

そして、いつも忘れる、半ページ戻る Ctrl+u
